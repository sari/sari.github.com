Install Tomcat and Java

  • Install Tomcat
  • Install Java
  • vi /home/tomcat/tomcat/bin/setenv.sh
        JAVA_HOME=/usr/lib/jvm/java-6-sun
    
    export CATALINA_OPTS="$CATALINA_OPTS -Xms128m -Xmx1024m -XX:MaxPermSize=256m"
  • vi /home/tomcat/tomcat/conf/server.xml
        # change the port 8080 to 8180.
  • Setup tomcat start/stop scripts as needed (sudo vi /etc/init.d/tomcat)
  • Increase memory in bin/catalina.sh accordingly: 
        JAVA_OPTS="-Xmx512M -Xms512M"

Sesame (RDF Triplestore) Installation

  • Download openrdf-sesame-2.6.3-sdk.tar.gz or get the latest version 
    sudo su - tomcat
    
    tar -xzvf openrdf-sesame-2.6.3-sdk.tar.gz
    
    cp openrdf-sesame-2.6.3/war/*.war /home/tomcat/tomcat/webapps/.
    
    # sudo cp openrdf-sesame-2.6.3/war/*.war /var/lib/tomcat7/webapps/.
    
    mkdir -p .aduna
    
    chmod 755 .aduna
  • Restart Tomcat, delete previous expanded war if upgrading
  • Your localhost Sesame Triplestore Repositories
  • Your localhost Sesame Triplestore System Information
  • Data Load

          Type: Native Java Store RDF Schema

    Click on "Browse", and find the emp_dept.rdf file you just downloaded, and then "Upload" button
