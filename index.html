  • shebang #! this setting in first line of python file will allow your code to execute without typing prefix "python"
    • Unix: #!/usr/bin/python -O
    • Windows Classroom 314: #!c:\Python27\python
    • Other Unix/Mac: #!/path/to/python
    • Other Windows: #!\path\to\python
    • Note: You can always prefix the executable file with "python" (python .\re.py)
  • Variations to execute a python file:
    • shebang is defined -> .\re.py
    • python is in the environment path -> python re.py
    • python is not in the environment path -> c:\Python27\python re.py
