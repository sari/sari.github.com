Git
 git add  start tracking new files and also to stage changes to already tracked files 
 git clone git@github.com:username/username.github.com.git  Pull a copy of a git repository to your local computer 
 git commit -m message  record your snapshot into your history 
 git init  create a repository from an existing directory of files 
 git pull  Pull the latest changes of a git repository to your local computer 
 git push origin master  update a remote repository with the changes you've made locally 
 git status -s  Get the current status of your local repository 
 ssh -T git@github.com  Test ssh key. Reply "yes" to continue connecting.

 
UNIX
 alias  list alias settings in ~/.bashrc 
 brew list  List brew installed applications 
 cd ~  change directory to your default (home) location 
 cd ..  change directory to one level up 
 cd ../..  change directory to two levels up 
 env  List environment variables 
 echo $PATH  List environment variable PATH setting 
 echo $SHELL  List current shell 
 edit ~/.profile  Edit profile to update PATH setting.
  Close/reopen terminal, or type "source ~/.profile" to reload profile 
 ls -al  list file in directory (a)all (l)long 
 mkdir abc  make a directory call abc 
 pip list  List pip installed packages 
 pwd  print current directory 
 rm -r abc  remove a directory (recursively). Use (f)forcefully for skipping questions. 
 sudo find / -name git  Locate all files/dir named git 
 touch filename  create an empty file called filename 
 whereis git  Locate where git is installed 
 which python  Locate current python environment 
 vad  (alias) activate virtual environment dev 
 da  (alias) deactivate current virtual environment 

 
Sample Configurations
 ~/.profile  # Configure prompt
 export PS1="[\@]\w $ "
 export EDITOR=mate
 export PATH=/usr/local/bin:/usr/local/share/python:$PATH 
 ~/.bashrc  # Aliases
 alias vad='. ~/.virtualenvs/dev/bin/activate'
 alias vd="deactivate"
