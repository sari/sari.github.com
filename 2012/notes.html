Emory University Libraries Python Group Notes

 Git Commands
 git init  create a repository from an existing directory of files 
 git status -s  Get the current status of your local repository 
 git add  start tracking new files and also to stage changes to already tracked files 
 git commit -m message  record your snapshot into your history 
 git push origin master  update a remote repository with the changes you've made locally 

 
UNIX Commands
 ls -al  list file in directory (a)all (l)long 
 mkdir abc  make a directory call abc 
 rm -r abc  remove a directory (recursively). Use (f)forcefully for skipping questions. 
 pwd  print current directory 
 cd ~  change directory to your default (home) location 
 cd ..  change directory to one level up 
 cd ../..  change directory to two levels up
