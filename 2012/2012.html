Meetings

  • 2012-01-09: Github Setup.
    • Create a new account on GITHUB, if you do not have one.
    • Install Apple's Xcode Developer Tools (free from App Store)
      • Check if git is installed
      • Open a terminal window ... Finder->Applications->Utilities->Terminal
      • type in "git" at the prompt.
      • If you do not get a response, then please manually install git using these instructions
    • Follow instructions to create a ssh key for GITHUB
    • Create a new repository to use as a user page
      • Go to the dashboard of your github account https://github.com/username and click on New Repository
      • Name this repository: username.github.com
      • Click on the newly created repository link and follow instructions to push a README file to this repository
      • Use sample index.html and style.css to get started
      • Don't forget to add a favicon using the free favicon generator!
